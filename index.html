Es un proyecto que permite medir y registrar variables climáticas utilizando sensores conectados a una placa Arduino. Este tipo de estación puede incluir sensores de temperatura, humedad y presión atmosférica. Los datos recolectados se pueden visualizar en timepo real a través de una pantalla LCD, o bien, se pueden enviar a una página web para su análisis y almacenamiento.
